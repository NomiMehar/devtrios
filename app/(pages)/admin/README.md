# Admin Dashboard

This directory contains the admin dashboard functionality for managing blogs.

## Structure

- `/login` - Admin login page
- `/dashboard` - Main admin dashboard
- `/dashboard/blogs` - Blog management (list, add, edit, delete)
- `/dashboard/blogs/new` - Create new blog
- `/dashboard/blogs/[id]/edit` - Edit existing blog

## Future Implementation

This structure is set up for future implementation of:
- Authentication and authorization
- Blog CRUD operations
- Image upload functionality
- Rich text editor for blog content
- Category management
